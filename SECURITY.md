# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 2.x     | :white_check_mark: |
| < 2.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in symflow, please report it through
[GitHub's private vulnerability reporting](https://github.com/vandetho/symflow/security/advisories/new).

**Please do not open a public issue for security vulnerabilities.**

You can expect an initial response within 48 hours. If the vulnerability is confirmed, a fix will be released as a patch version as soon as possible.
